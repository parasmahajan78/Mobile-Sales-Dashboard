# 📱 Mobile Sales Dashboard (Power BI)

A comprehensive Power BI dashboard designed to analyze mobile phone sales performance across different cities, brands, payment methods, and time periods. This dashboard provides key business insights through interactive visualizations and KPI metrics.

## 🚀 Project Overview

The Mobile Sales Dashboard helps businesses monitor and evaluate sales trends, customer preferences, product performance, and payment behaviors. It enables data-driven decision-making through intuitive and interactive reports.

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)
- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales Value

### Sales Analysis
- Monthly Sales & Quantity Trends
- City-wise Sales Distribution
- Brand-wise Sales Performance
- Top Selling Mobile Models

### Customer Insights
- Customer Ratings Analysis
- Day-wise Customer Activity

### Payment Analysis
- UPI Transactions
- Debit Card Transactions
- Credit Card Transactions
- Cash Payments

### Interactive Filters
- Month-wise Slicer
- Brand Filter
- Payment Method Filter
